## Настройка Telegram бота

### Шаг 1: Создание бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Придумайте имя для бота (например: "KenigSwap Support Bot")
4. Придумайте username для бота (например: "kenigswap_support_bot")
5. BotFather отправит вам **токен** бота - сохраните его!

Пример токена: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`

### Шаг 2: Получение Chat ID

Есть несколько способов получить Chat ID:

#### Способ 1: Через бота @userinfobot
1. Найдите бота [@userinfobot](https://t.me/userinfobot)
2. Отправьте ему любое сообщение
3. Он ответит вам с вашим Chat ID

#### Способ 2: Через API
1. Отправьте любое сообщение вашему новому боту
2. Откройте в браузере:
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
3. Найдите в ответе поле `"chat":{"id":...}` - это ваш Chat ID

### Шаг 3: Настройка переменных окружения в Supabase

1. Откройте [Supabase Dashboard](https://supabase.com/dashboard)
2. Выберите ваш проект
3. Перейдите в **Settings → Edge Functions → Secrets**
4. Добавьте два секрета:

```
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
TELEGRAM_CHAT_ID=ваш_chat_id
```

**ВАЖНО**: Не используйте кавычки, просто вставьте значения как есть.

### Шаг 4: Проверка

1. Откройте страницу `/support` на вашем сайте
2. Заполните форму обратной связи
3. Нажмите "Отправить сообщение"
4. Вы должны получить сообщение в Telegram!

## Troubleshooting

### Сообщения не приходят в Telegram

1. **Проверьте токен бота**:
   ```bash
   curl https://api.telegram.org/bot<ВАШ_ТОКЕН>/getMe
   ```
   Должен вернуть информацию о боте

2. **Проверьте Chat ID**:
   - Убедитесь, что вы отправили хотя бы одно сообщение боту
   - Chat ID должен быть числом (может быть отрицательным)

3. **Проверьте секреты в Supabase**:
   - Settings → Edge Functions → Secrets
   - Оба секрета должны быть добавлены
   - Нет лишних пробелов или кавычек

4. **Проверьте логи Edge Function**:
   - Edge Functions → send-telegram-message → Logs
   - Смотрите ошибки в консоли

### Ошибка "Missing Telegram credentials"

Это означает, что переменные окружения не настроены в Supabase:
1. Откройте Supabase Dashboard
2. Settings → Edge Functions → Secrets
3. Добавьте `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`

### Ошибка "Failed to send message to Telegram"

Возможные причины:
1. Неверный токен бота
2. Неверный Chat ID
3. Вы не отправили боту первое сообщение (бот не может писать первым)
4. Бот был заблокирован

### Форма показывает ошибку при отправке

1. Откройте консоль браузера (F12)
2. Проверьте ошибки в сети (вкладка Network)
3. Убедитесь, что Edge Function развернута
4. Проверьте, что env переменные настроены в `.env` файле:
   ```
   NEXT_PUBLIC_SUPABASE_URL=...
   NEXT_PUBLIC_SUPABASE_ANON_KEY=...
   ```

## Альтернативный способ: Отправка в группу

Если хотите получать сообщения в группу, а не в личный чат:

1. Создайте группу в Telegram
2. Добавьте вашего бота в группу
3. Сделайте бота администратором
4. Получите Chat ID группы (начинается с `-`)
5. Используйте этот Chat ID вместо личного
