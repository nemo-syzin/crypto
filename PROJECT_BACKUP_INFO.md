## Основные компоненты

### Структура проекта
- **Frontend:** Next.js 14 с TypeScript
- **Стилизация:** Tailwind CSS + shadcn/ui
- **База данных:** Supabase
- **3D эффекты:** Three.js + Vanta.js
- **Анимации:** Framer Motion

### Ключевые страницы
- `/` - Главная страница с героем и основными разделами
- `/exchange` - Калькулятор обмена криптовалют
- `/rates` - Страница курсов криптовалют с данными CoinGecko
- `/about` - О компании с интерактивными элементами
- `/blog` - Блог с новостями и аналитикой
- `/support` - Поддержка с FAQ и чатом
- `/dashboard` - Личный кабинет пользователя
- `/operator-dashboard` - Панель оператора для чата
- `/policy/*` - Политики (AML/KYC, конфиденциальность, условия)

### API эндпоинты
- `/api/rates` - Получение курсов обмена
- `/api/coingecko` - Прокси для CoinGecko API
- `/api/exchange-orders` - Создание заявок на обмен
- `/api/bestchange-feed` - XML фид для BestChange

### База данных (Supabase)
- `kenig_rates` - Курсы обмена валют
- `exchange_orders` - Заявки на обмен
- `user_profiles` - Профили пользователей
- `chat_sessions` - Сессии чата
- `chat_messages` - Сообщения чата
- `chat_operators` - Операторы поддержки

### Основные функции
1. **Обмен криптовалют** - Полнофункциональный калькулятор с поддержкой множества валютных пар
2. **Система курсов** - Интеграция с CoinGecko API для актуальных данных
3. **Аутентификация** - Полная система регистрации/входа через Supabase Auth
4. **Чат поддержки** - Реальное время общение с операторами
5. **Адаптивный дизайн** - Полная поддержка мобильных устройств
6. **3D визуализация** - Интерактивные элементы на страницах

### Конфигурация
- Все переменные окружения настроены через `.env.example`
- Поддержка развертывания на Netlify
- Оптимизация производительности и SEO

## Инструкции по восстановлению

Если нужно восстановить эту версию:

1. Убедитесь, что все файлы соответствуют структуре, описанной выше
2. Установите зависимости: `npm install`
3. Настройте переменные окружения согласно `.env.example`
4. Запустите миграции Supabase
5. Запустите проект: `npm run dev`
