# Настройка SEO и Google Search Console

## Что уже сделано

1. **Favicon** - ваш логотип с короной теперь отображается во вкладке браузера
2. **Open Graph метатеги** - для правильного отображения в соцсетях и поисковиках
3. **Twitter Card метатеги** - для Twitter/X
4. **Структурированные данные (JSON-LD)** - помогает Google понять структуру сайта
5. **Sitemap.xml** - карта сайта для поисковых систем (доступна по адресу `/sitemap.xml`)
6. **Robots.txt** - правила для поисковых роботов

## Как настроить Google Search Console

### Шаг 1: Верификация домена

1. Перейдите на https://search.google.com/search-console
2. Нажмите "Добавить ресурс" → "Домен"
3. Введите `kenigswap.ru`
4. Google предложит несколько способов верификации:
   - **DNS** (рекомендуется) - добавьте TXT запись в настройках вашего домена
   - **HTML файл** - загрузите файл в папку `/public`
   - **HTML тег** - добавьте мета-тег в `<head>`

### Шаг 2: Отправка Sitemap

После верификации:

1. В боковом меню выберите "Sitemap"
2. Добавьте URL: `https://kenigswap.ru/sitemap.xml`
3. Нажмите "Отправить"

### Шаг 3: Запрос индексации

1. В Search Console используйте "Проверка URL"
2. Введите `https://kenigswap.ru`
3. Нажмите "Запросить индексацию"
4. Повторите для важных страниц:
   - `/exchange`
   - `/rates`
   - `/about`

## Когда увидите логотип в Google?

- **Обычно**: 1-2 недели после индексации
- **Для ускорения**:
  - Убедитесь что сайт уже проиндексирован
  - Запросите повторную индексацию через Search Console
  - Проверьте что `NEXT_PUBLIC_SITE_URL` в `.env` указывает на правильный домен

## Проверка метатегов

Используйте эти инструменты для проверки:

1. **Open Graph**: https://www.opengraph.xyz/
2. **Twitter Card**: https://cards-dev.twitter.com/validator
3. **Rich Results Test**: https://search.google.com/test/rich-results
4. **Schema Markup Validator**: https://validator.schema.org/

## Важные файлы

- `/app/layout.tsx` - основные метатеги и Open Graph
- `/app/sitemap.ts` - генератор sitemap
- `/public/robots.txt` - правила для роботов
- `/public/логотип.svg` - ваш логотип (используется в Open Graph)
- `/.env` - переменная `NEXT_PUBLIC_SITE_URL`
